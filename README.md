Los proyectos están ubicados dentro de la carpeta `src` de este repositorio. Para poder ejecutarlos de forma local en tu computadora, es necesario contar con los siguientes requisitos previos instalados:

* **Node.js** (Se recomienda la versión 24.16.0 o superior).
* **Angular CLI** instalado de forma global.
