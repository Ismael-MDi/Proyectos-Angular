# Proyectos en Angular: Suma y Factorial

**Autor:** Ismael Morales Diaz
**Ubicación:** Los proyectos principales se encuentran dentro de la carpeta `src`.

## Requisitos Previos
* **Node.js** (Versión 24.16.0 o superior).
* **Angular CLI** (Instalado de forma global).

## Instrucciones de Ejecución

1. **Instalar dependencias:** Abre la terminal en la carpeta raíz del proyecto y ejecuta el comando:
`npm install`

2. **Iniciar el servidor:** Una vez instalados los módulos, levanta el entorno de desarrollo con:
`ng serve`

3. **Ver la aplicación:** Abre tu navegador web e ingresa a la siguiente dirección para interactuar con los proyectos:
`http://localhost:4200`
